## 4. Objetivos
- General: Desarrollar el sitio web corporativo con formulario de contacto funcional y seguro.
- Específicos: Implementar secciones informativas, API de contacto con validación y rate limit, y preparar despliegue en Vercel.

## 5. Marco Teórico
Referencias a Next.js, React, Tailwind, seguridad web (CSP, HSTS), validación y A11y.

## 6. Metodología
Proceso iterativo por semanas (7-12): requisitos, desarrollo, pruebas, documentación y despliegue.

## 7. Desarrollo
- Secciones frontend (hero, about, services, contact, footer).
- API de contacto con validación/sanitización y rate limiting.
- Proxy de seguridad con headers y CORS.
- Documentación y scripts de verificación.

## 8. Resultados
- Sitio responsive con formulario conectado al backend.
- Metadata SEO, sitemap y robots.
- Seguridad básica aplicada.
